## ✨ Sobre o Projeto

O **V1CTOR HUB** não é apenas um "Linktree". É uma experiência visual imersiva projetada para converter visitantes em clientes. Com uma estética baseada em **Glassmorphism** e **Dark Mode Premium**, o hub centraliza portfólio, serviços de automação e canais de contato de forma elegante e performática.

Este projeto foi concebido e estruturado inteiramente através do **Google AI Studio**, demonstrando o poder da colaboração entre humanos e Inteligência Artificial no desenvolvimento de interfaces modernas.

## 🛠️ Tecnologias Utilizadas

- **Core**: [Next.js 15](https://nextjs.org/) (App Router) & [React 19](https://react.dev/)
- **Estilização**: [Tailwind CSS 4.0](https://tailwindcss.com/) para um design fluido e moderno.
- **Animações**: [Framer Motion](https://www.framer.com/motion/) para micro-interações e transições suaves.
- **Ícones**: [Lucide React](https://lucide.dev/) para uma interface limpa e intuitiva.
- **Inteligência**: Integrado com [Google Gemini API](https://ai.google.dev/) para funcionalidades inteligentes.

## 🌟 Diferenciais Visuais

- 🌌 **Ambient Backgrounds**: Gradientes pulsantes e blur dinâmico que criam profundidade.
- 💎 **Glassmorphism**: Efeito de vidro jateado em todos os cards e containers.
- ⚡ **Micro-interações**: Feedback tátil em cada clique e hover, incluindo efeitos de brilho (shine) nos botões de destaque.
- 📱 **Ultra-Responsivo**: Experiência perfeita do mobile ao desktop.
- 🎨 **Tipografia Dinâmica**: Elementos flutuantes de background que reforçam a identidade de "Automation" e "Artificial Intelligence".

## 🚀 Como Executar Localmente

### Pré-requisitos
- Node.js (v18 ou superior)
- NPM ou Yarn

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/SeuUsuario/V1CTOR-HUB.git
   cd V1CTOR-HUB
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env.local` na raiz do projeto e adicione sua chave da API do Gemini:
   ```env
   GEMINI_API_KEY=sua_chave_aqui
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
   Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
